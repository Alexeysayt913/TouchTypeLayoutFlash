
# TouchTypeLayoutFlash

TouchTypeLayoutFlash is a small helper tool for touch typists. It displays a large popup notification in the center of the screen whenever the keyboard layout changes (EN, RU, UA, etc.). This helps you stay aware of the active input language and avoid typing in the wrong layout.

---

## Features

- **Large centered popup** when switching keyboard layout
- **Supports any languages** (EN, RU, UA, and others)
- **Works on Windows**, written in [AutoHotkey](https://www.autohotkey.com/)
- Especially useful for **touch typists** who do not look at the tray icon

---

## Installation and Usage

1. Install [AutoHotkey v1.x](https://www.autohotkey.com/) (if not already installed).

---

## Why This Tool Exists

When touch typing, it is easy to miss the moment when the keyboard layout switches. TouchTypeLayoutFlash solves this problem by showing a clear, noticeable popup every time the input language changes.

---

## Customization

You can modify the script to adjust:

- Popup size and position
- Colors and transparency
- Font
- Display duration

All settings are located directly in the script and are easy to edit.
